## Overview

**Venezuela Super Lawyer** is a comprehensive AI-powered legal operating system specialized in Venezuelan law. It provides doctoral-level expertise across multiple legal domains, enabling:

- Comprehensive legal research across Venezuelan legal sources
- Constitutional analysis with OEA vs SAPI text comparison
- TSJ (Tribunal Supremo de Justicia) outcome prediction with ML
- Automated legal document generation
- Law and Resolution drafting with implementation roadmaps
- Legislative feasibility and voting requirement mapping

```bash
# ╔═══════════════════════════════════════════════════════════════════════════╗
# ║                        INSTALLATION GUIDE                                   ║
# ╚═══════════════════════════════════════════════════════════════════════════╝

git clone https://github.com/rikitrader/venezuela-super-lawyer.git
cd venezuela-super-lawyer

# Step 2: Create Virtual Environment (Recommended)
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Step 3: Install Dependencies
pip install -r requirements.txt

# Step 4: Configure Environment
cp .env.example .env
# Edit .env with your configuration

# Step 5: Set Access Key (REQUIRED)
export VSL_ACCESS_KEY="your_secure_password_here"

# Step 6: Verify Installation
python3 scripts/security.py verify

# Step 7: Run Tests
python3 -m pytest tests/ -v
```
