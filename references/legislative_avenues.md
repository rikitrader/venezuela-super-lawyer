# Venezuelan Legislative Avenues Reference Guide

## Overview of Normative Instruments in Venezuela

This reference documents all available legal avenues to create binding rules in Venezuela, their constitutional basis, procedural requirements, and publication rules.

---

## 1. LEY ORDINARIA (Ordinary Law)

### Constitutional Basis
- **CRBV Arts. 202-218** — Legislative process
- **CRBV Art. 187** — National Assembly competencies

### Characteristics
| Attribute | Description |
|-----------|-------------|
| **Issuing Authority** | Asamblea Nacional |
| **Hierarchy** | Below Constitution, above regulations |
| **Subject Matter** | Any matter within AN competence |
| **Required Majority** | Simple majority |
| **Publication** | Gaceta Oficial (mandatory) |

### Process Flow
```
1. INICIATIVA (Art. 204 CRBV)
   → Diputados AN (3+)
   → Ejecutivo Nacional
   → TSJ (materias judiciales)
   → CNE (materia electoral)
   → Ciudadanos (0.1% electores)
   → Consejos Legislativos Estadales

2. PRIMERA DISCUSIÓN
   → Comisión permanente estudia
   → Informe a plenaria
   → Debate general
   → Aprobación en principio

3. SEGUNDA DISCUSIÓN
   → Artículo por artículo
   → Enmiendas y modificaciones
   → Votación articulada
   → Aprobación definitiva

4. SANCIÓN
   → Presidente AN firma
   → Remisión al Ejecutivo

5. PROMULGACIÓN
   → Presidente República promulga
   → O devuelve con observaciones

6. PUBLICACIÓN
   → Gaceta Oficial de la República
   → Entrada en vigencia
```

### Timeframes (Typical)
- First discussion to second: 15+ days
- Presidential promulgation: 10 days (or 20 if urgent)
- Total average: 2-6 months (can be faster with urgency)

## 2. LEY ORGÁNICA (Organic Law)

### Constitutional Basis
- **CRBV Art. 203** — Definition and requirements

### When Required
1. Leyes que la Constitución denomina así
2. Leyes que desarrollan derechos constitucionales
3. Leyes que sirven de marco normativo a otras leyes
4. Leyes de organización de poderes públicos

### Special Requirements
| Attribute | Description |
|-----------|-------------|
| **Majority** | 2/3 de los presentes |
| **Control** | Sala Constitucional TSJ must certify organic character |
| **Pre-admission** | TSJ ruling on constitutionality (Art. 203 último aparte) |

### Process Flow
```
Same as ordinary law, PLUS:

PASO ADICIONAL (antes de promulgación):
→ Remisión a Sala Constitucional TSJ
→ Pronunciamiento sobre carácter orgánico
→ Si no es orgánica → devuelta como ley ordinaria
```

## 3. DECRETO CON RANGO, VALOR Y FUERZA DE LEY (Decree-Law)

### Constitutional Basis
- **CRBV Art. 236.8** — Presidential attributions
- **CRBV Art. 203** — Enabling law requirements

### Requirements
1. **Ley Habilitante** — Enabling law passed by AN with 3/5 majority
2. **Tiempo limitado** — Must be within enabling period
3. **Materia específica** — Only on matters delegated

### Process Flow
```
1. SOLICITUD DE HABILITACIÓN
   → Presidente solicita a AN
   → Especifica materias y plazo

2. LEY HABILITANTE
   → AN aprueba con 3/5 votos
   → Define alcance, materias, plazo
   → Publica en Gaceta Oficial

3. ELABORACIÓN DEL DECRETO
   → Presidente redacta
   → Consulta ministerial
   → Firma del Decreto

4. PUBLICACIÓN
   → Gaceta Oficial
   → Vigencia inmediata o diferida

5. CONTROL
   → AN puede derogar/modificar después
   → TSJ puede anular si excede habilitación
```

### Speed Advantage
- Can be issued in days if enabling law exists
- Bypasses full legislative debate

## 4. REGLAMENTO (Executive Regulation)

### Constitutional Basis
- **CRBV Art. 236.10** — Reglamentar total o parcialmente las leyes

### Characteristics
| Attribute | Description |
|-----------|-------------|
| **Issuing Authority** | Presidente de la República |
| **Hierarchy** | Below law it implements |
| **Subject Matter** | Implementation details of existing law |
| **Limitation** | Cannot exceed, contradict, or modify the law |
| **Publication** | Gaceta Oficial (mandatory) |

### Types
1. **Reglamento Ejecutivo** — Implements specific law
2. **Reglamento Autónomo** — Organizational matters (limited scope)
3. **Reglamento Delegado** — When law explicitly delegates

### Process Flow
```
1. IDENTIFICAR LEY BASE
   → Verificar ley permite reglamentación
   → Verificar no existe reglamento vigente

2. ELABORACIÓN TÉCNICA
   → Ministerio competente redacta
   → Consulta jurídica (Procuraduría)
   → Revisión Consejo de Ministros

3. APROBACIÓN
   → Consejo de Ministros aprueba
   → Presidente firma

4. PUBLICACIÓN
   → Gaceta Oficial
   → Vigencia según disposición final
```

## 5. RESOLUCIÓN (Ministerial/Agency Resolution)

### Constitutional/Legal Basis
- **CRBV Art. 226** — Ministerial competencies
- **LOAP Art. 49** — Administrative acts

### Characteristics
| Attribute | Description |
|-----------|-------------|
| **Issuing Authority** | Ministros, entes descentralizados, reguladores |
| **Hierarchy** | Below reglamentos |
| **Subject Matter** | Technical, operational, sector-specific |
| **Publication** | Gaceta Oficial if general; notification if particular |

### Types
1. **Resolución de carácter general** — Affects indeterminate persons
2. **Resolución particular** — Affects specific person/entity
3. **Resolución conjunta** — Multiple ministers

### Process Flow
```
1. COMPETENCIA
   → Verificar atribución legal del ministerio/ente
   → Verificar no invade reserva legal

2. ELABORACIÓN
   → Consulta técnica interna
   → Informe jurídico
   → Firma del Ministro/Autoridad

3. PUBLICACIÓN (si es general)
   → Gaceta Oficial
   → Principio de publicidad normativa

4. NOTIFICACIÓN (si es particular)
   → Notificación personal
   → Recursos administrativos habilitados
```

## 6. ORDENANZA (Municipal Ordinance)

### Constitutional Basis
- **CRBV Art. 168-182** — Municipal competencies
- **LOPPM** — Ley Orgánica del Poder Público Municipal

### Characteristics
| Attribute | Description |
|-----------|-------------|
| **Issuing Authority** | Concejo Municipal |
| **Hierarchy** | Below national laws; above municipal regulations |
| **Subject Matter** | Local competencies (Art. 178 CRBV) |
| **Publication** | Gaceta Municipal |

### Municipal Competencies (Art. 178 CRBV)
- Ordenación urbanística
- Servicios públicos locales
- Turismo local
- Tránsito urbano
- Espectáculos públicos
- Cementerios
- Arquitectura civil

### Process Flow
```
1. INICIATIVA
   → Alcalde
   → Concejales
   → Ciudadanos (iniciativa popular)

2. PRIMERA DISCUSIÓN
   → Comisión del Concejo
   → Debate general

3. SEGUNDA DISCUSIÓN
   → Articulado
   → Aprobación

4. PROMULGACIÓN
   → Alcalde promulga (o veta)
   → Si veta, Concejo puede insistir con 2/3

5. PUBLICACIÓN
   → Gaceta Municipal
   → Entrada en vigencia
```

## 7. ENMIENDA CONSTITUCIONAL

### Constitutional Basis
- **CRBV Arts. 340-341**

### Scope
- Adición o modificación de uno o varios artículos
- SIN alterar estructura ni principios fundamentales

### Process Flow
```
1. INICIATIVA (Art. 341.1)
   → 15% ciudadanos inscritos en RE
   → 30% diputados AN
   → Presidente en Consejo de Ministros

2. TRÁMITE PARLAMENTARIO
   → AN tramita según su Reglamento
   → Aprobación por mayoría

3. REFERENDO (Art. 341.3)
   → CNE convoca referendo
   → Aprobada si mayoría vota SÍ

4. PROMULGACIÓN
   → Presidente promulga
   → Publicación en Gaceta Oficial
```

### Limitations
- NO puede afectar:
  - Estructura fundamental del Estado
  - Principios constitucionales esenciales
  - Derechos humanos fundamentales

---

## 8. REFORMA CONSTITUCIONAL

### Constitutional Basis
- **CRBV Arts. 342-346**

### Scope
- Revisión parcial de la Constitución
- Sustitución de cláusulas
- SIN afectar estructura y principios fundamentales

### Process Flow
```
1. INICIATIVA (Art. 342)
   → 15% ciudadanos inscritos en RE
   → Mayoría diputados AN
   → Presidente en Consejo de Ministros

2. PRIMERA DISCUSIÓN AN
   → Período de sesiones de presentación

3. SEGUNDA DISCUSIÓN AN
   → Período de sesiones siguiente
   → Aprobación por 2/3 de diputados

4. REFERENDO (Art. 344)
   → 30 días después de aprobación AN
   → Aprobada si mayoría vota SÍ

5. PROMULGACIÓN
   → Presidente promulga en 10 días
   → Publicación Gaceta Oficial
```

---

## 9. ASAMBLEA NACIONAL CONSTITUYENTE

### Constitutional Basis
- **CRBV Arts. 347-350**

### Scope
- Transformar el Estado
- Crear nuevo ordenamiento jurídico
- Redactar nueva Constitución

### Process
```
1. INICIATIVA (Art. 348)
   → Presidente en Consejo de Ministros
   → 2/3 de diputados AN
   → 2/3 de Concejos Municipales
   → 15% de electores inscritos

2. CONVOCATORIA
   → Referendo para convocar ANC

3. ELECCIÓN
   → Elección de constituyentes

4. FUNCIONAMIENTO
   → ANC con poderes originarios
   → Límites solo derivados del pueblo

5. APROBACIÓN
   → Referendo aprobatorio de nueva Constitución
```

## Comparative Table: Speed, Risk, and Requirements

| Avenue | Speed | Legal Risk | Required Majority | Publication | Typical Use |
|--------|-------|------------|-------------------|-------------|-------------|
| Ley Ordinaria | Medio (2-6 meses) | Bajo | Simple | Gaceta Oficial | Materias generales |
| Ley Orgánica | Lento (3-12 meses) | Bajo | 2/3 + TSJ | Gaceta Oficial | Derechos, organización |
| Decreto-Ley | Rápido (días-semanas) | Medio | N/A (habilitación previa) | Gaceta Oficial | Emergencias, habilitación |
| Reglamento | Rápido (semanas) | Bajo-Medio | N/A | Gaceta Oficial | Implementación |
| Resolución | Muy rápido (días) | Medio | N/A | Gaceta/Notif. | Técnico, sectorial |
| Ordenanza | Medio (1-3 meses) | Bajo | Simple/2/3 | Gaceta Municipal | Local |
| Enmienda | Lento (6+ meses) | Alto | Mayoría + Referendo | Gaceta Oficial | Cambio puntual CRBV |
| Reforma | Muy lento (12+ meses) | Muy alto | 2/3 + Referendo | Gaceta Oficial | Cambio parcial CRBV |

## Publication Requirements (Principio de Publicidad Normativa)

### General Rule
Toda norma de carácter general debe ser publicada en la Gaceta Oficial para entrar en vigencia.

### Specific Requirements

| Type | Publication Venue | Entry into Force |
|------|-------------------|------------------|
| Ley | Gaceta Oficial República | Fecha indicada o publicación |
| Decreto-Ley | Gaceta Oficial República | Fecha indicada o publicación |
| Reglamento | Gaceta Oficial República | Fecha indicada o publicación |
| Resolución General | Gaceta Oficial República | Fecha indicada o publicación |
| Resolución Particular | Notificación personal | Notificación |
| Ordenanza | Gaceta Municipal | Fecha indicada o publicación |

### Constitutional Amendments/Reforms
- Publication: Gaceta Oficial Extraordinaria
- Entry: Upon publication (post-referendum)
